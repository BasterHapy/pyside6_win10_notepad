## Qtdesigner

## Win10_notepad
这个下边存放的是运行文件以及UI界面文件

### 如何运行

确保你已经安装了`Pyside6`

```bash
pip3 install pyside6
```
克隆项目到本地
```bash
git clone https://github.com/BasterHapy/pyside6_win10_notepad.git
```
运行
```bash
python3 win10_notepad/win10_notepad_L4_run.py 
```
